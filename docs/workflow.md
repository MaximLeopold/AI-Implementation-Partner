# AI Implementation Partner Workflow

## Stage 1 — Shared Understanding

Use Grill Me.

Goal:

* Understand the idea.
* Explore design decisions.
* Reach shared understanding.

Output:

* Shared understanding recap.

## Stage 2 — Planning

Use Planning and Task Breakdown.

Goal:

* Create implementation plan.
* Define phases.
* Identify dependencies.

Output:

* Implementation roadmap.

## Stage 3 — Build

Use TDD.

Goal:

* Build incrementally.
* Validate behavior.

Output:

* Working implementation.

## Stage 4 — Handoff

Use Handoff.

Goal:

* Transfer project state.
* Enable another agent or engineer to continue work.

Output:

* Project continuation document.
